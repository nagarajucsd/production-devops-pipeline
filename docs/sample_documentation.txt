Moduele2: Build a Java Spring boot Application

step1: create a spring boot project from https://start.spring.io/ with dependencies (Spring Web, Boot Actuator, boot devtools) and download the zip and extract it in the project folder /app
step2: open the application in vscode (code .) and run the application 
step3: create rest controller inside src/main/java/com/devops/springbootapp folder /controller file HomeController.java
step4: test it (http://localhost:8081) and and check whether it is health ans its version also
step5: maven build package (creating the jar file)
step6: adding the changes in project folder and pushing it to the github

errors faced:
1. encountered with error port 8080 in use 
sol : tried in deleting the process id related to port 8080 again and again without checking what's the error is exactly then realised that  it(8080) already assigned to jenkins so changed the spring boot port to 8081 (In appliacation.properties: server.port=8081)
    and restarted the application
2. 404 Not Found No static resource health for request '/health'  
sol: configured acutation in application.properties   management.endpoints.web.exposure.include=health,info  --> This tells Spring Boot which Actuator endpoints should be accessible over HTTP.
                                                      management.endpoint.health.show-details=always  -->  This controls how much information the /actuator/health endpoint returns.
enabled mapping logs ::  logging.level.org.springframework.web.servlet.mvc.method.annotation.RequestMappingHandlerMapping=TRACE --> tells Spring Boot to log every URL mapping that it registers when the application starts.
 also in pom.xml added the dependency 
  (<dependency>
    <groupId>org.springframework.boot</groupId>         --> It turns your Spring Boot project into a web application 
    <artifactId>spring-boot-starter-web</artifactId>         by adding everything needed to build REST APIs and web applications.
  </dependency>)                                             spring-boot-starter-web is a bundle of web-related libraries.
